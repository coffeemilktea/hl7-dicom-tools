Morph rules

Rules are applied in order to top-level tags. set creates the tag if missing; remove deletes it; new UID regenerates the UID consistently across all files (identical old UIDs map to identical new UIDs, preserving study/series linkage).

Apply & export

Export serializes the in-memory dataset (inline edits + applied rules). Multiple files download as a ZIP. Compressed pixel data is preserved as-is; the transfer syntax is never transcoded.

Supported SOP Classes

Coverage of the storage SOP classes supported by the RadiAnt DICOM viewer. image renders through the pixel pipeline (any other storage SOP class carrying standard Image Pixel module data will also render), video plays via the browser's media engine, pdf displays the encapsulated document, tags opens in the tag browser / morph editor only.
